The Treblle Dashboard provides you with a unified user interface for monitoring and observing your API [projects](/platform/projects/).

You have instant access to the dashboard after <a href="https://app.treblle.com/register" target="_blank">creating a Treblle account</a> or visiting your <a href="https://app.treblle.com">dashboard</a> if you already have an account.

In the dashboard you see a list of all your [projects](/platform/projects/), get access to your API key which you will need for integrating a Treblle SDK in your codebase and so much more.

## Accessing your API key

You API keys is one of two pieces needed to integrate your codebase with a Treblle SDK, the second piece is your [Project ID](/platform/projects/#project-id)

To locate your API key, in the Treblle Dashboard, on the top right hand of the navigation, you will find your profile avatar. Click on the avatar to get a dropdown and then click on **Account Settings**.

> If you are logged in already in your Treblle Dashboard, you can click on <a href="https://app.treblle.com/users/profile">this link</a> to access your account settings

Once you are on your account settings page, you will find your API key at the top, simply click on the **Copy key** button to copy your API key.

## Widgets

Widgets are a fundamental way to understand your API data. API users want different information about their APIs depending on their role.

Here are the Widgets we currently provide:

### Top cities

List of top cities from which users access your API.

### Top countries

List of top countries from which users access your API.

### Number of Requests

Provides an overview of how many requests you get every day.

### Requests map

View recent requests made to your API directly on a live map.

### Top devices

List of top devices that are used to access your API.

### Client App versions

Shows the versions of applications or browsers that accessed your API

### Average Load Time

The average load time on your API.

### Average response size

The average response time on your API.

### Recent Requests List

Provides a summary of recent requests made to your API.

### Performance

Provides a graphical view of request load times for a user-specified time period.


## Removing widgets

You can remove a widget by clicking on the x button on the top-right corner of the widget card.
Doing so will remove the widget from your dashboard.
