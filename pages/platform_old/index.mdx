## Features

By integrating Treblle to your API, you get the following features out of the box:

- Real-time API monitoring and logging
- Auto-generated [API documentation](/platform/api-documentation/) with <a href="https://swagger.io/specification/" target="_blank">Open API Specification</a> support
- [API analytics](/platform/api-analytics/)
- [API governance scoring](/platform/api-governance/)
- [API security](/platform/api-security/)
- One-click testing
- API management on the go
- [Share request](/platform/requests/#share-request)
- [Error reporting](/platform/problems/)

## Getting Started

To get started with Treblle, <a href="https://app.treblle.com/register" target="_blank">sign up</a> for a free account. Then check the [integrations](/integrations/) page to find the Treblle SDK for the platform your API is built on. We've covered most of all the popular backend languages and frameworks out there.

Found a missing integration or your language/framework is not in the docs? Please suggest it on Discord <a href="https://treblle.com/chat">#integration-ideas channel</a> and even consider adding it yourself on GitHub.

The Treblle Docs is an open source project and contributions from developers like yourself are welcome!

Stay in tune with your APIs!
